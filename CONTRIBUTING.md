# 参与贡献

感谢你帮助微信年轮变得更准确、更安全。提交代码即表示你有权提供该内容，并同意按项目 MIT License 发布。

## 开始开发

```bash
npm install
npm run dev:desktop
```

提交前运行：

```bash
npm run check
```

## 最需要的贡献

- 不含真实隐私的导出格式样本与字段说明
- 不同微信版本/系统的兼容性验证
- 大文件性能、时区和金额统计测试
- 无障碍、中文文案和年报视觉改进
- 独立适配器接口设计与安全评审

## 数据样本规则

Issue、PR、测试夹具只能使用人工构造的数据。不得提交真实联系人、聊天内容、账单、数据库、密钥、手机号、邮箱、头像或设备路径。

## 统计规则

- 新统计必须说明数据来源、可检测范围和缺失条件。
- 未导入不能显示为真实 0。
- 删除与撤回不得根据猜测补齐。
- 支付金额优先使用官方账单，并排除无效状态。

## 适配器规则

涉及进程读取、密钥或数据库解密的实验不得直接加入主应用。请先讨论独立适配器、许可证、系统安全和本人授权边界，遵守 `docs/SECURITY-BOUNDARY.md`。
