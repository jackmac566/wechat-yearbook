# 更新日志

所有版本从真实发布记录开始，不补写不存在的历史版本。

## V0.2.0 — 2026-07-22

双平台开源桌面预览版。

### 年报体验

- 默认进入一屏一个结论的沉浸式年报故事，降低原仪表盘的信息密度。
- 保留详细总览、聊天、关系、朋友圈与账单页面。
- 未导入的数据明确显示“未导入”，不拿零值冒充真实统计。

### 兼容与准确性

- 适配多种大小写字段、EchoTrace/WeFlow 风格字段和嵌套会话 JSON。
- 新增带 `schemaVersion` 的微信年轮标准 JSON 导入与导出。
- 修正本地时区边界下活跃日与连续活跃天数的计算。
- 新增兼容字段与标准格式往返测试。

### 开源与桌面发行

- 增加 Mac Apple 芯片、Mac Intel 与 Windows 的 GitHub Actions 自动打包。
- 增加 MIT License、第三方参考、隐私、安全、贡献、架构、数据格式和新手发布文档。
- 固定 npm 依赖版本并强化 Electron 的导航、沙箱与网页安全边界。
- 主程序不包含微信进程扫描、密钥提取和数据库解密；未来只能由可移除的独立适配器接入。

## V0.1.2 — 2026-07-21

真实数据与上传向导版。

### 真实

- 首页默认改为空白数据状态，未导入就不显示统计数字。
- 演示数据只在用户主动点击后出现，并持续标明为模拟记录。
- 清除导入后回到空白状态，不再自动恢复演示数据。

### 向导

- 增加微信支付个人对账 CSV 的手机申请、解压与上传步骤。
- 增加 Windows 与 macOS 聊天记录迁移、开源导出和文件选择说明。
- 增加朋友圈可获取范围、字段格式与不完整性提示。

### 适配与边界

- 适配 `wechat-export-macos` 的 `chat.json` 与 `chat.csv` 字段，正确识别“我”发出的消息、消息类型与撤回记录。
- 明确区分官方账单、第三方本地导出和实验性朋友圈数据。
- 明确提示 macOS 重签微信的安全影响；不建议普通用户直接操作，不建议关闭 SIP。
- 彻底删除且无残留的数据不会被恢复、估算或伪造。

## V0.1.1 — 2026-07-21

公开发行准备版。

### 新增

- 增加全访客可见的独立更新日志。
- 侧边栏常驻当前版本入口。
- 更新离线缓存版本，确保访客及时获得新版。

### 优化

- 失败、退款、退还、撤销和关闭交易不计入有效金额。
- 优先使用消息 ID 与微信交易单号去重。
- 公开分享卡继续默认隐藏联系人、聊天原文和金额。

## V0.1.0 — 2026-07-21

首个可用体验版。

### 核心

- 年度消息、活跃日、连续活跃与聊天时段分析。
- 文字、图片、语音、视频、表情、文件、链接与拍一拍分类。
- 联系人和群聊发送热度排行。

### 数据

- 支持聊天、朋友圈 JSON / CSV 导入。
- 支持微信支付个人对账 CSV，统计红包和转账。
- 撤回、删除和朋友圈均显示可检测范围与可信度。

### 体验

- 完成 PWA 桌面安装与离线访问。
- 本机生成年度分享卡片。
- 不要求微信密码，不上传原始聊天文件。
