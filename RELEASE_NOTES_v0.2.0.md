# 微信年轮 V0.2.0

首个可直接公开到 GitHub 的双平台桌面源码版本。

## 新增

- 默认进入简洁的沉浸式年度故事，一屏展示一个结论
- 保留详细总览、聊天轨迹、关系热度、朋友圈与红包转账面板
- 兼容更多 EchoTrace/WeFlow 风格字段、大小写字段和嵌套会话 JSON
- 支持带版本号的微信年轮标准 JSON 导入/导出
- GitHub Actions 自动构建 Mac Apple 芯片、Mac Intel 与 Windows 安装包
- MIT 许可、隐私、安全、贡献、架构、数据格式与小白发布文档

## 安全与真实性

- 没有导入的数据明确显示“未导入”，不把 0 当真实结果
- 取密钥与数据库解密不进入主程序；不要求关闭 macOS SIP
- 修正本地时区边界下活跃日与连续天数计算
- 分享图默认不含联系人、聊天原文和具体金额

## 安装提示

为了保持 0 成本，本版本没有购买 Apple/Windows 代码签名。请只从本仓库 Release 下载，并核对 Actions 构建记录。Mac 第一次打开可在 Finder 中按住 Control 点应用 → 打开；不要关闭 SIP。
